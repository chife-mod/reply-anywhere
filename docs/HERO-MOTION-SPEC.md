## Поведение (спек клиента + правки утра 2026-07-02)

1. **Орбитальное движение.** По трём орбитам плавно летают круги с логотипами
   каналов. Каждая орбита — со своей скоростью (чуть разной).
   **Состав: 14 каналов, без дубликатов** (клиент обновил макет 2026-07-02:
   добавлены Walmart, eBay, Etsy; дубли Instagram/Messenger убраны; синк 1:1
   с Figma 31:2808 — tiktok, appstore, messenger, walmart, instagram,
   trustpilot, whatsapp, facebook, amazon, google-maps, etsy, ebay, google,
   google-play).
2. **Пульс свечения.** Свечение за центральным логотипом плавно пульсирует:
   чуть больше → чуть меньше → … (очень плавно). Реализация: opacity/scale
   пред-размытых слоёв, НЕ анимировать blur.
   **Постоянного halo под каналом (Ellipse 854 у Amazon) НЕТ** — в макете это
   была лишь демонстрация вида свечения в момент события; свечение канала
   появляется только как вспышка прилёта/ответа.
3. **Связь (рандомизированно, ЖИВО):**
   a. Проявляется **рельса** — градиентная 1px полоска от центра к кругу канала
      (затухает у обоих концов, как в макете).
   b. По рельсе летит **импульс** (два размытых круга): у круга канала появляется
      маленьким и прозрачным → растёт, непрозрачнеет → улетает **под** центральный
      тайл.
   c. В момент касания тайла — glow тайла **чуть ярче**, деликатно (scale ~1.12,
      не bloom).
   d. **Короткий бит** (~0.3–0.4s, не секунда) — и импульс летит **обратно**;
      при прибытии вспышка у канала **компактная** (~1.5× диска, не огромная).
   e. Рельса затухает и исчезает.
   f. **Параллельность:** одновременно живут до 3 связей к разным каналам
      (стаггер-запуск, общий set занятых каналов — двух связей к одному каналу
      не бывает). Паузы между циклами короткие (0.25–1.15s).
4. **Z-порядок (КРИТИЧНО):** рельсы и импульсы — **ниже** кругов с логотипами.
   Импульсы никогда не пролетают ПОВЕРХ логотипов.
   Слои снизу вверх: glow-квадраты → орбиты → рельсы+импульсы → круги каналов →
   центральный тайл.

## Обязательные практики

- transform/opacity-only; никакого анимируемого `filter: blur` (слои пред-размыты).
- Пауза, когда hero вне вьюпорта (IntersectionObserver) и когда вкладка скрыта.
- `prefers-reduced-motion: reduce` → полностью статичная сцена (состояние макета).
- GSAP + MotionPathPlugin (орбиты) — уже в `vite.optimizeDeps`.
