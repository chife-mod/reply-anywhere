## Phase 0 — Фундамент и стек · 🔵 · ✅ ГОТОВО (2026-07-01)

Технический хребет, независимый от дизайна.
- Astro SSG + CF Workers Static Assets, build зелёный.
- AI-SEO слой (robots allow-list, JSON-LD, llms.txt, sitemap).
- Демо-форма end-to-end на Worker + Turnstile + CF Analytics (env-gated).
- Типизированные Content Collections.
- **Выход:** живой быстрый скаффолд, куда «наливается» дизайн. Детали — `STACK.md`.

## Phase 1 — Передача дизайна · 🟣 · ⬅ МЫ ЗДЕСЬ

Олег финализирует Figma и отдаёт вход для реализации.
- **Вход:** Figma уже есть (по словам заказчика).
- **Нужно:** ссылка на Figma-файл/фреймы (desktop + mobile), секции по BRIEF §5
  (hero → логотипы → фичи/мультиканальность → 3 кейса → демо-форма), фиолетовый SF-стиль.
  Ассеты от Ajax (скрины продукта) + логотипы (Ajax, Samsung, Volia).
- **Ворота → Phase 2:** дизайн утверждён, ссылка/фреймы доступны Claude.

---

## Phase 2 — Реализация фронтенда (Figma → код) · 🔵

Перенос дизайна в этот стек, секция за секцией.
- Через Figma MCP вытянуть контекст: размеры, цвета, шрифты, переменные.
- **Сверить `tokens.css` с Figma-переменными** (не изобретать — брать из дизайна).
- Собрать каждую секцию как Astro-компонент, заменяя placeholder'ы.
- Реальный копирайт (из gominga, упрощённый) → в `data/landing.json`.
- Адаптив (mobile + desktop) строго по Figma.
- **Ворота → Phase 3:** все секции совпадают с Figma, build зелёный, ноль ошибок консоли.

---

## Phase 3 — Движение и медиа · 🔵

Скорость + «вау», без потери производительности.
- Нативные CSS scroll-driven анимации по умолчанию; GSAP-island только где нужна
  оркестровка таймлайна.
- `astro:assets` (AVIF/WebP, responsive srcset, lazy); hero-медиа через R2 / CF Images.
- `prefers-reduced-motion` уважается.
- **Ворота → Phase 4:** LCP-медиа оптимизированы, анимации не роняют INP на mobile Safari.

---

## Phase 4 — SEO / AI / контент финализ · 🔵 (+🟣 копирайт)

- Реальная OG-картинка (1200×630).
- Финал JSON-LD (+ FAQPage, если появится FAQ-копирайт), синк `llms.txt`, мета по секциям.
- Финальные тексты подтверждены.
- **Ворота → Phase 5:** SEO-аудит чистый, мета/schema валидны.

---

## Phase 5 — Деплой и запуск · 🔵 (+🟣 креды/домен)

- Создать CF Turnstile + Web Analytics; проставить секреты (`TURNSTILE_SECRET`,
  `LEAD_WEBHOOK_URL` → SF HelpDesk) через `wrangler secret put`.
- GitHub Actions: build + деплой на CF Workers, preview-URL на каждый PR.
- Замеренный Lighthouse + perf-budget в CI (число вместо обещания).
- Домен **www.replyanywhere.com** → Cloudflare, HSTS, go-live чеклист.
- **Выход:** публичный быстрый лендинг на проде.
