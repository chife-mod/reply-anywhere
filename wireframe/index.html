Эмоция

«Вау, сколько всего» → ясность: мы начинаемся там, где обычные review-tools заканчиваются.

Визуал

Облако channel-pills вокруг 2-строчного заголовка. Цвет группирует: тёмные = review-mgmt, акцентные = что добавляет Reply Anywhere. Затем сворачивается в центр.

Motion

Scroll-scene: плитки влетают (сначала review, потом все), заголовок собирается по частям → на скролле всё исчезает, заголовок поднимается, появляются описание + 3 фичи. GSAP ScrollTrigger; reduced-motion → статичный collapsed.

Связь

Ядро wedge (перекликается с §04/§07); поглощает прежнюю channels-секцию. Фичи-трио дублирует §02 — свести к одному.
