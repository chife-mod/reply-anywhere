04Why

Core wedge vs review-only tools

Смысл

Главное отличие: конкуренты = только отзывы; мы = отзывы + соцсети/личка из коробки (на стеке Semantic Force).

Эмоция

«Ага, вот в чём разница» — момент ясности.

Визуал

comparison-table, two-column, check-vs-cross. Конкурента не называем.

Motion

Правая колонка-«победитель» подсвечивается при скролле; галочки появляются по очереди.

Связь

Аргумент после доверия (§03); §06 затем доказывает охват фактически.

Метафора

Они закрывают одну дверь — мы открываем все.

Перевод (RU)

Заголовок: Не только отзывы. Отзывы и соцсети.

Тело: Большинство инструментов ограничиваются отзывами в сторах. Reply Anywhere отвечает везде, где реально говорят клиенты — отзывы и соцсети, в одном месте.
